# PM2.5 Exposure and Deprivation in London: A Spatial Analysis

Code and figures for 5SSG2060 Principles of Spatial Data Science Coursework 2.

## Contents
- `5SSG2060_CW2.ipynb` — Python notebook containing all analysis code
- Figures 1-5 as PNG files

## Data Sources
- PM2.5 data: Millington (2024) london-data repository
- IMD 2019: MHCLG via London Datastore
- LSOA boundaries: Millington (2024) london-data repository

## Dependencies
- geopandas
- libpysal
- esda
- splot
- scipy
- matplotlib
